# aws profile switcher

## install

* download the files and rename to the cmd you'd prefer
* place under your path e.g. `/usr/local/bin/awsp`

## usage

```bash
~/Code/warnerbros
🍻 awsp
ask
cnd
cnmsd
cnmsp
cnp
```

```bash
~/Code/warnerbros
🍺awsp cnd
~/Code/warnerbros [cnd-us-west-2]:
🌨exit

~/Code/warnerbros
🍻
```
